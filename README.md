# 使い方
1. [足す小説]と[引く小説]それぞれに小説の題名を記入する。
    * 片方の欄が空白のままでも大丈夫。
    * 複数ある場合は、全角または半角スペースで区切る。

2. 1.での計算の結果に似ている順に小説が表示される。
    * 表右側に表示される数値は**類似度**。

##### 例
* 足す小説 : `こころ 銀河鉄道の夜`
* 引く小説 : `星の王子さま`

こう書くと、『こころ』+『銀河鉄道の夜』-『星の王子さま』という **小説の計算** が行われる。

##### 活用例
* 好きな小説を5冊挙げ、全て[足す小説]に書く。→次の1冊が見つかるかも？？
